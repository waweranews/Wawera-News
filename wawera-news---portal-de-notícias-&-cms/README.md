This website is built with React and integrated with **Decap CMS** (formerly Netlify CMS) for easy content management.

### 📝 Managing Content

*   **Posts (News):** Create, edit, or delete news articles. You can set featured posts and add markdown-formatted body content.
*   **Portfolio:** Manage your featured work. You can update logos, descriptions, and WhatsApp links.
*   **Clients:** Add or update partner logos. Use the "Featured" toggle to highlight key partners with a gradient border.
*   **Settings:** Update global site information like the slogan, meta description, and brand colors.

### 🛠 Technical Stack

*   **Frontend:** React 18 + Vite
*   **Styling:** Tailwind CSS
*   **CMS:** Decap CMS (Git Gateway)
*   **Hosting:** Netlify
*   **Animations:** Motion (Framer Motion)

### 📁 Project Structure

*   `/admin`: CMS interface and configuration.
*   `/src/content`: JSON/Markdown files where content is stored.
*   `/src/components`: Reusable UI components.
*   `/src/pages`: Main application pages.
